 Limitations:

- The network speed maximum is about 100 MBPS. Don't try to install VPN Sofware on VM (this will not work, exclude VPN extension on browser can be used).

- If the main network disconnect or not work, VM will shut down (about 60s) or you must re-run the script again (input n and press enter when prompted).
